
## Let's Start

You need to have JDK, Maven, and Intellij IDEA(preferred)


## Run project

Clone the project

```bash
  git clone hhttps://github.com/ADA-GWU/2024-a2-cicd-alizadeht/
```

Go to the project directory

```bash
  for example:

  cd 2024-a2-cicd-alizadeht
```

Install dependencies

```bash
  npm install
```

Start the testing

```bash
  npm test
```
